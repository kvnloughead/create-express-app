# Mongo-Express (mxp)

Command line utility to streamline creation of Express projects using MongoDB.

## Usage

`mxp -n project-name`

## Development Usage

I've been setting an alias 

```bash
alias runmxp='cd ~/dev/testdir && rm -rf project && mxp'
```

And then I run 

```bash
runmxp -n project
```

to test it.
