# Mongo-Express (mxp)

Command line utility to streamline creation of Express projects using MongoDB.

## Usage

`create-express-app -n project-name`

## Development Usage

I've been setting an alias 

```bash
alias run-cea'cd ~/dev/testdir && rm -rf project && create-express-app'
```

And then I run 

```bash
run-cea -n project
```

to test it.
